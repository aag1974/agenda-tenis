Manual do coach

Este manual mostra como o coach usa o Tennis Flow Scouting para coordenar a marcação de partidas de múltiplos atletas em um torneio, delegar a marcação para outros atletas (scouters) e acompanhar os resultados em tempo real.

1Como funciona — em 3 etapas

📝
Coach cria a partida
Seleciona o atleta e gera um link
→
📲
Scouter marca
Recebe o link, marca pontos ao vivo
→
📊
Coach acompanha
Vê placar, stats e relatório

O coach tem login e gerencia tudo no dashboard. O scouter é qualquer pessoa (outro atleta da equipe, um assistente) que abre o link no celular e marca a partida — não precisa de cadastro nem de senha.

2Login

Acesse tennis-flow.com.br/scouting no celular ou computador. Use as credenciais que o Tennis Flow forneceu.

Dica: salve a senha no navegador na primeira vez — próximos acessos são instantâneos. No celular, adicione à tela de início para abrir como um aplicativo.

3Dashboard

Após o login, o dashboard mostra duas áreas:

➕
Nova partida

Para criar uma nova partida: seleciona o atleta, opcionalmente o adversário, e gera o link para o scouter.

📋
Partidas

Lista de todas as partidas criadas, com status atualizado automaticamente (a cada 5 segundos).

Status das partidas

  • AGUARDANDO

    Link enviado, scouter ainda não abriu.

  • ● AO VIVO

    Scouter está marcando pontos agora.

  • ENCERRADO

    Partida finalizada normalmente.

  • RET

    Partida encerrada por desistência (Retirement).

4Criar uma nova partida

  1. Selecionar o atleta. Digite parte do nome no campo Atleta. A lista filtra conforme você digita. Toque no nome para selecionar.
  2. Selecionar o adversário (opcional). O segundo campo é opcional. Se preencher, a lista já vem filtrada pela mesma categoria do atleta. Se deixar em branco, o scouter completa antes de iniciar.
  3. Nome fora da lista. Se o atleta não está cadastrado, basta digitar o nome completo — ao apertar Enter ou sair do campo, o nome é aceito como livre.
  4. Gerar link pro scouter. Toque no botão. O cartão da partida aparece na lista abaixo com status AGUARDANDO.

5Mandar o link para o scouter

No cartão da partida, dois botões compartilham o link:

📋
Link scouter

Copia o link para a área de transferência. Você pode colar em qualquer mensageiro ou aplicativo.

💬
WhatsApp

Abre o WhatsApp Web/App com o link e uma mensagem pronta. Você escolhe o contato e envia.

Importante: o link só funciona uma vez. Quando o scouter abre e clica em "Iniciar tracking", a partida fica vinculada a ele. Para outra partida, gere um novo link.

6O que o scouter vê

Ao abrir o link no celular, o scouter encontra uma tela de configuração com:

  • Atleta pré-preenchido

    Nome que você selecionou — o scouter não pode mudar.

  • Adversário

    Se você preencheu antes, já vem selecionado. Se não, o scouter escolhe da lista (filtrada pela mesma categoria do atleta).

  • Quem começa sacando

    Atleta ou adversário.

  • Formato

    Melhor de 3, Melhor de 3 com super-TB no 3º set, 1 set + match TB, ou Pro-set de 8 games.

  • Vantagem

    Ad (regra normal) ou No Ad (no 40-40 o próximo ponto decide).

Após clicar em "▶ Iniciar tracking", o scouter começa a marcar pontos.

7Como o scouter marca os pontos

A tela é dividida em dois lados de cor: azul = atleta, vermelho = adversário. Cada ponto é classificado pelo tipo de jogada:

Saque (do servidor da vez)

Ace · Service Winner · 1st Serve Fault · Double Fault

Devolução (de quem recebe)

Return Winner · Return Error · Return in Play

Bola em jogo

Winner · Forced Error · Unforced Error

Ações secundárias

Desfazer último · → Game manual · ⏹ Encerrar

Game manual

Se o scouter perdeu a marcação dos pontos de um game (distração, atraso pra começar), ele usa "→ Game manual" para marcar quem venceu o game sem precisar inventar pontos. O placar é atualizado normalmente.

8Encerrar uma partida

A partida pode ser encerrada de três formas:

Naturalmente (placar fecha)

Quando o atleta ou adversário fecha o resultado conforme o formato escolhido (ex: 2 sets em melhor de 3). O sistema marca como ENCERRADO automaticamente.

Retirement (alguém desistiu)

Se alguém parou de jogar durante a partida (lesão, problema físico), o scouter clica em ⏹ Encerrar e seleciona quem parou. A partida vai para RET e o outro lado fica como vencedor.

Pelo coach (scouter sumiu)

Se o scouter parou de marcar mas a partida continuou, o coach pode encerrar pelo dashboard. Toca em Encerrar no cartão da partida e escolhe entre:

  • Marcar placar final — informa o placar dos sets manualmente.
  • Retirement — indica quem parou.
W.O. (não compareceu). Se o atleta não jogou, não use Encerrar — exclua a partida pelo botão Excluir no cartão. W.O. significa que a partida não rolou; não tem placar para registrar.

9Acompanhar uma partida ao vivo

Enquanto o scouter marca, o coach vê o status atualizar automaticamente no dashboard a cada 5 segundos (sem precisar dar refresh). No cartão da partida AO VIVO, o botão Acompanhar abre uma tela em modo somente leitura, mostrando:

  • Placar em tempo real — sets, games, pontos.
  • Stats acumuladas — Aces, Winners, Erros, etc.
  • Momentum — gráfico mostrando o fluxo de pontos.
  • Notas de performance — qualitativas, do scouter ou do coach.

O coach não marca pontos nessa tela — só o scouter pode. Mas o coach pode adicionar notas (durante e depois do jogo).

10O que as stats significam

  • Nota Técnica · 0-10

    Indicador agregado calculado pelos winners, erros e eficiência de saque. Quanto maior, melhor a performance técnica.

  • 1st Serve % / 1st Serve Pts Won

    Percentual de primeiros saques que entraram e percentual de pontos ganhos com o primeiro saque.

  • 2nd Serve Pts Won

    Pontos ganhos quando o primeiro saque falhou.

  • Break Points Won

    Quebras de saque conquistadas dividido por chances criadas. Ex: 3/8 = 3 quebras em 8 oportunidades.

  • Winners / Unforced Errors

    Pontos fechados em bola boa (winner) vs erros não-forçados (decisões ruins, bolas fáceis perdidas).

11Notas de performance

A qualquer momento durante o jogo, o scouter pode adicionar observações que os números não capturam — uma percepção tática, um sinal emocional, um padrão técnico. As notas ficam vinculadas ao placar do momento.

Cada nota tem uma categoria opcional:

Técnico

Saque, devolução, forehand, backhand, voleio, etc.

Tático

Posicionamento, escolha de bola, padrão de jogo.

Físico

Cansaço, lesão, hidratação, ritmo.

Emocional

Confiança, frustração, foco, agressividade.

Depois que a partida encerra, o coach pode continuar adicionando notas pelo link Ver relatório — útil para a análise pós-jogo.

12Relatório final

Quando a partida encerra, o cartão na lista mostra o placar dos sets (ex: 7-6 · 6-7 · 6-3) e o botão Ver relatório. O relatório reúne tudo:

  • Placar final com sets, vencedor marcado;
  • Stats completas (acima);
  • Momentum (gráfico de fluxo de pontos);
  • Notas de performance acumuladas (ao vivo e pós-jogo).

O relatório tem um link permanente. Você pode compartilhar com o atleta, pais ou outros coaches — eles abrem em modo somente leitura.

Excluir uma partida. Se criou um link por engano ou a partida não vai acontecer, use o botão Excluir no cartão. O link vira inválido para o scouter.
